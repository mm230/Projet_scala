# Projet_scala
Le but de ce projet est de créer une application de gestion des emplois du temps et des notes en utilisant le langage de programmation Scala.

Nous avons débuté le projet par créer une base de données qui pourrait accueillir les différentes données généréés dans l'application.

Pour l'api, nous avons utiliser Play framework et nous l'avons connecté avec notre projet en utlisant sbt. 

Mais par soucisde delais et de compréhension du mécanisme de fonctionnement d'une API, nous n'avons pas pu terminé le travail.

Il faut dire que beaucoup de ces concepts que nous devions utiliser dans ce projet, sont totalement nouveau pour nous. Et donc il nous a fallu beaucoup de temps pour pouvoir comprendre leurs mécanismes de fontionnement. Nous avons fait de notre mieux pour essayer de rendre un travail cohérent avec les attentes du projet, mais force est de constater que ce n'est pas le cas. 
Nous nous portons entierement responsables de ce que nous avons produit et assumons pleinement les sanctions qui incombent au projet.


Cordialement.
